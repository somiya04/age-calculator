# Age Calculator 🎂

An interactive Age Calculator that tells you exactly how old you are — down to the day!

## Features
- 🎯 Exact age in years, months, and days
- 🎂 Birthday countdown — how many days until your next birthday
- 📅 Day of the week you were born on
- 🌍 Fun facts about your birth year
- 😊 Emoji based on your age group
- 🎊 Confetti animation on calculate
- 🫧 Floating bubble background effect
- 🎉 Special birthday message if today is your birthday!

## Built With
HTML | CSS | JavaScript (No libraries used!)

## Live Site
https://somiya04.github.io/age-calculator

## How to Use
1. Select your date of birth
2. Click Calculate
3. See your age, fun facts, and enjoy the confetti! 🎊
